========================================
  איך להוסיף פרויקטים לאתר
  How to add projects to the website
========================================

עברית:
------
1. צור תיקיה חדשה בתוך תיקיית "projects"
   שם התיקיה = שם הפרויקט שיופיע באתר
   
   לדוגמה: "משרדי גוגל תל אביב"

2. הכנס תמונות לתיקיה (JPG, PNG, WEBP, GIF)
   התמונות יופיעו בסדר אלפביתי לפי שם הקובץ

3. הפעל את run-build.bat (לחיצה כפולה)
   זה יעדכן את הקובץ projects.js אוטומטית

4. רענן את האתר בדפדפן - הפרויקט החדש יופיע!


English:
--------
1. Create a new folder inside the "projects" folder
   Folder name = Project name that will appear on the website
   
   Example: "Google Offices Tel Aviv"

2. Put images in the folder (JPG, PNG, WEBP, GIF)
   Images will appear in alphabetical order by filename

3. Run run-build.bat (double-click)
   This automatically updates the projects.js file

4. Refresh the website in your browser - the new project will appear!
